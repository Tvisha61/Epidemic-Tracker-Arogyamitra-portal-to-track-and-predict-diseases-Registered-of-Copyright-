# 🩺 Epidemic Tracker: Arogyamitra Portal  
**(© Registered Copyright)**

Arogyamitra is a smart disease tracking and prediction portal designed to help health authorities and citizens monitor, analyze, and respond to epidemic outbreaks efficiently. Built using **Flask**, the platform combines real-time data, analytics, and prediction models to enable proactive public health management.

---

## 🔍 Overview

Arogyamitra serves as a digital companion for epidemic control and early warning. With dynamic mapping, user symptom submissions, and AI-powered prediction models, the platform empowers healthcare officials with actionable insights to contain outbreaks.

---

## ⚙️ Tech Stack

**Backend:**  
- Python (Flask)  
- Flask-Restful / Flask-SQLAlchemy  
- Pandas, Scikit-learn (for analytics/prediction)  
- SQLite or MySQL  

**Frontend:**  
- HTML5, CSS3, Bootstrap  
- JavaScript (Chart.js / Leaflet for maps)  
- Jinja2 templating  

**Data Visualization & Prediction:**  
- Matplotlib, Seaborn  
- Machine learning models for prediction  
- Heatmaps & Graphs for tracking trends  

---

## 💡 Key Features

- 📝 **User Symptom Reporting** – Collect symptom data from citizens
- 🗺️ **Heatmap & Zone Tracker** – View affected zones and intensity
- 📊 **Data Visualization** – Daily/weekly infection trend reports
- 🧠 **Disease Prediction Model** – Forecast potential outbreak risks
- 📬 **Automated Alerts** – Notification system for high-risk zones
- 🔒 **Secure Admin Panel** – Verified access for health officers
